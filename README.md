# LLVM-Tutorial
## Setting Environment
```bash
# in ubuntu 20:04
$ apt-get install -y llvm-15 llvm-15-dev llvm-15-tools clang

## Run
$ clang++ -g -O3 [source file] `/usr/lib/llvm-15/bin/llvm-config --cxxflags --ldflags --libs` -o [output name] -Xlinker --export-dynamic
```
